WrandR
======

aRandr for wayland.

![screenshot](./screenshot.png)



## Install

```bash
$ go get -v -u github.com/csicar/wrandr
$ ~/go/bin/wrandr
$ # if you want to run it as a application
$ cp wrandr.desktop ~/.local/share/applications/
```
